# MSFT Incident Report + IP Form

This is a merged GitHub Pages build for the Mary Seacole Foundation Trust simulation activity.

## Pages settings

Use GitHub Pages with:

- Branch: `main`
- Folder: `/docs`

## Flow

1. Students open `docs/index.html` and complete the incident report.
2. After submission, the final button links to `./ip-form.html`.
3. Students then complete the Incidents in Practice form.

The root `index.html` and `ip-form.html` are included as duplicates for easy local testing.
